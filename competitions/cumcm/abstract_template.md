# 摘要模板 (5 段式 abstract_template)

> 国赛高分摘要通常接近 5 段式。Stage 8 最后一步，从 `workspace/output/final/traceability.md` 允许进入摘要的 claim 和 `workspace/output/final/final_results.md` 中抽取关键信息填入下列占位符。字数 600-900，必须包含可追溯的定量结果。

---

## 5 段式骨架

### 段 1 — 背景与问题定义 (100-150 字)

```
本文针对【核心问题描述,30字】, 综合考虑【关键约束/因素 1】、【因素 2】和【因素 3】, 
建立了【模型族命名变体, 不要 vanilla 名】模型, 通过【求解方法/算法名】求解, 
得到了【可量化的总成果概述, 含至少 1 个定量】。
```

**填空要点**:
- 核心问题描述: 用题目中的核心动词 (如"调度"、"配比"、"预测") 而非空泛"研究"
- 因素 1-3: 关键约束或主要考量，应来自 `workspace/output/q*/assumptions.md`
- 模型族命名变体: 必须包含修饰词 (如"动态权重"、"改进型"、"混合的", 见 winning_patterns §4)
- 求解方法: 算法名 + 主要工具 (e.g., "Lagrangian 松弛 + cvxpy")
- 总成果: ≥1 个数 (利润、误差、时间)

### 段 2 — 任务拆解 (50-100 字)

```
本文针对问题一、问题二、问题三, 分别建立了【模型 1 简称】、【模型 2 简称】、【模型 3 简称】。
其中, 问题三利用问题一所得【关键结果】作为输入, 进一步【新动作】。
```

**填空要点**:
- 三个模型简称尽量统一在一个族系内 (winning_patterns §3 模型族协调)
- 强调 Q3 复用 Q1/Q2 (winning_patterns §5 复用链, 关键加分)

### 段 3 — 各小问详细方法 + 数值结果 (250-400 字, 主体)

```
针对问题一, 本文【动作: 建立/构造/采用】【模型 1 名】, 通过【求解技术】求解, 
得到【定量结果 1, 必须有数】, 相比【基线/常规方法】提升【百分比】。

针对问题二, 本文在问题一基础上, 引入【新约束/新因素】, 升级为【模型 2 名】, 
得到【定量结果 2】, 表明【物理意义】。

针对问题三, 本文【动作】, 求解得【定量结果 3】, 满足【约束验证】。
```

**填空要点**:
- 每问至少 1 个数 (合计 ≥3, anti_pattern A1)
- 每问 1 个动词 + 1 个名词的组合, 避免重复"研究/分析/讨论"
- 与基线对比 (winning_patterns §3 加分点)

### 段 4 — 灵敏度 + 模型评价 (80-120 字)

```
对模型进行了【方法 LHS / Sobol】的多变量联合灵敏度分析, 
在【关键参数】在 ±X% 范围内变化时, 输出指标偏差小于 Y%, 验证了模型的稳健性。
进一步识别出当【临界参数】超过【阈值】时, 模型出现【相变特征】, 
该发现为决策提供【实际指导】。
```

**填空要点**:
- 必须出现 "多变量联合" / "LHS" / "Sobol" 之一 (winning_patterns §7)
- 给具体百分比, 不要 "较为稳健"
- 包含临界参数发现 (一等奖加分)

### 段 5 — 创新与推广 (60-100 字)

```
本文创新点在于:
① 【创新点 1】;
② 【创新点 2】;
③ 【创新点 3】。
所建模型可进一步推广至【场景 1】和【场景 2】等领域。
```

**填空要点**:
- 创新点 1-3 来自 `workspace/output/final/final_results.md`、`q*_summary.md` 和验证/灵敏度结论
- 推广必须 ≥2 具体场景, 不能 "可推广至各类相关问题" (anti_pattern H3)

### 关键词

```
关键词: 【核心问题, 4-8 字】, 【主方法 1】, 【主方法 2】, 【分析手段】, 【应用领域】
```

**填空要点**:
- 4-6 个 (anti_pattern A4)
- 第 1 个: 题目核心问题
- 第 2-3 个: 主方法 (e.g., "Lagrangian 松弛", "拉丁超立方")
- 第 4 个: 分析手段 (e.g., "灵敏度分析")
- 第 5 个: 应用领域 (e.g., "生产调度")
- 每个 ≤8 字

## 自检 7 项 (写完摘要必查)

- [ ] 共 5 段, 顺序无误
- [ ] 字数 600-900 (anti_pattern A5)
- [ ] 段 1 出现命名变体 (winning_patterns §4)
- [ ] 段 3 共有 ≥3 个具体定量数值 (anti_pattern A1)
- [ ] 段 4 出现 "多变量联合" / "LHS" / "Sobol" 之一
- [ ] 段 5 创新点 ≥3 + 推广 ≥2 具体场景
- [ ] 关键词 4-6 个, 每个 ≤8 字, 无空泛词

## 完整示例 (供模仿)

> 本文针对生产调度中**多产品-多时段-随机需求**的最优分配问题, 综合考虑预算约束、
> 库存能力和需求不确定性, 建立了**基于 Lagrangian 松弛的混合整数线性规划**模型, 
> 通过 cvxpy + GUROBI 求解, 得到了相比贪心基线提升 12.3% 的最优产销方案。
> 
> 本文针对问题一、问题二、问题三, 分别建立了**确定性 MILP**、**含库存约束的扩展 MILP**
> 和**两阶段随机规划**模型。其中, 问题三利用问题一所得最优产量 $x_1^*$ 作为初始 warm start, 
> 进一步引入需求随机扰动。
> 
> 针对问题一, 本文构造 Lagrangian 松弛对偶问题, 通过次梯度法迭代求解, 
> 得到最优产量 $x^* = (12, 0, 25, ...)$, 总利润 87234 元, 相比贪心基线提升 12.3%。
> 针对问题二, 本文在问题一基础上引入库存上限 $K \leq 100$ 件, 升级为含 220 个 0-1 决策变量的扩展 MILP, 
> 得到调度方案利润 82150 元, 表明**库存能力是利润的瓶颈, 而非预算**。
> 针对问题三, 本文构造两阶段场景树 (200 个场景), 求得稳健最优期望利润 78920 元, 
> CVaR-95% 风险值 5230 元, 满足风险预算约束。
> 
> 对模型进行了基于**拉丁超立方采样**的多变量联合灵敏度分析, 
> 在 $(p, c, B)$ 在 ±10% 范围内变化时, 总利润偏差小于 7.8%, 验证了模型的稳健性。
> 进一步识别出当预算 $B$ 减少 30% 以上时, 最优产品组合发生显著切换 (Hamming 距 > 50%), 
> 该发现表明 $B = 70k$ 是模型的相变点, 为决策者预留预算缓冲提供量化依据。
> 
> 本文创新点在于:
> ① 提出 Lagrangian 松弛与 warm start 复用的双向耦合求解架构;
> ② 设计了多档 LHS 联合扰动方案,定量给出三档稳健区间;
> ③ 识别出模型的预算相变点,为风险预算提供了实践指导。
> 所建模型可进一步推广至**物流配送中心选址优化**和**电网调度**等领域。
> 
> **关键词**: 生产调度, Lagrangian 松弛, 混合整数规划, 拉丁超立方抽样, 鲁棒优化
