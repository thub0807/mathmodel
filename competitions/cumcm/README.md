# CUMCM Assets

This directory contains reusable CUMCM-oriented reference assets for `mathmodel-md-copilot`.

## Use In v1.2

Use these files progressively:
- Stage 2 planning support: naming ideas, domain patterns, structure hints
- Stage 8 paper generation support: abstract templates, phrase banks, paper skeletons
- Stage 9 final review support: anti-pattern checks and presentation review

## Main Assets

| File | Typical use |
|---|---|
| `winning_patterns.md` | planning, writing, and structure inspiration |
| `anti_patterns.md` | final review and cleanup |
| `phrase_bank.md` | writing support |
| `abstract_template.md` | paper abstract drafting |
| `paper_skeleton.md` | paper structure drafting |
| `distilled_*.md` | optional targeted writing and naming support |
| `empirical.json` | optional background evidence notes for style or density expectations |

## Legacy or Secondary Assets

- `topic_specs.json` is a legacy competition-structure asset and is not part of the active `v1.2-alpha` workflow.
- `rubric_overlay.json` may remain useful as background judging context, but it is not a required active rubric input.

## Important Boundary

These assets do not control startup, topic-letter selection, or multi-topic scoring in the active workflow. They are optional support material for planning, writing, and review.
