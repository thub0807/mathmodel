## 电工杯与国赛的核心差异

1. **题量更大**: 通常 6-8 子问, 国赛 3-5。stage 5 时间预算需放大
2. **学科聚焦**: 电力 / 能源 / 控制 / 电力市场 / 新能源, 题目情境工程化
3. **数据驱动**: 题目通常给出大量 Excel/CSV 实测数据 (电网负荷 / 光伏出力 / 故障记录), 数据预处理重要
4. **物理意义优先**: 电工杯评审更看重"模型的物理意义是否清晰"而非"创新性"
5. **工程实用性**: 推荐方案需评估"工程上可实现", 不能只是数学最优
6. **无官方 LaTeX 模板**: 各队自由排版, 字体 / 排版规范松, 但仍要求 GB/T 7714
7. **页数 25-30**: 略多于国赛, 含工程图表
8. **阅读对象**: 评委多为电力 / 能源行业资深工程师, 论文需"工程师视角可读"

## 推荐使用方式

电工杯当前最大风险是**子问数多**导致 stage 5 时间超支。建议:
- 优先 fast 模式跑通 1 遍, 看哪些 Qi 是"主线", 哪些是"加分项"
- 主线 Qi 用 standard 精修; 加分 Qi 走 carryover, stage 9 终审再补
- 每 Qi 严控 1.5h 预算, 超时强制进下一题
