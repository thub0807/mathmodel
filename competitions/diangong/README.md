# Diangong Assets

This directory contains reusable Diangong-oriented reference assets for `mathmodel-md-copilot`.

## Use In v1.2

Use these files progressively:
- Stage 2 planning support
- Stage 8 paper generation support
- Stage 9 final review support

## Main Assets

| File | Typical use |
|---|---|
| `winning_patterns.md` | planning and writing inspiration |
| `anti_patterns.md` | final review and cleanup |
| `phrase_bank.md` | engineering-style Chinese phrasing support |
| `abstract_template.md` | abstract drafting |
| `paper_skeleton.md` | paper structure drafting |
| `empirical.json` | optional background note, currently seed-level |

## Current Evidence Status

This directory is still `seed v0.1`. Treat it as useful domain support rather than as a fully distilled competition corpus.

## Legacy or Secondary Assets

- `topic_specs.json` is a legacy asset and is not part of the active `v1.2-alpha` workflow.
- `rubric_overlay.json` may remain as optional review context, not as an active control file.

## Important Boundary

These assets do not serve as a startup questionnaire, topic-letter selector, or multi-topic scoring mechanism in the active workflow.
