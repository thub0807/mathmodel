# MCM and ICM Assets

This directory contains reusable MCM and ICM-oriented reference assets for `mathmodel-md-copilot`.

## Use In v1.2

Use these files progressively:
- Stage 2 planning support
- Stage 8 paper generation support
- Stage 9 final review support

## Main Assets

| File | Typical use |
|---|---|
| `winning_patterns.md` | planning and writing inspiration |
| `anti_patterns.md` | final review and cleanup |
| `phrase_bank.md` | English paper phrasing support |
| `abstract_template.md` | summary and letter drafting |
| `paper_skeleton.md` | paper structure drafting |
| `empirical.json` | optional background note, currently seed-level |

## Current Evidence Status

This directory is still `seed v0.1`. Its assets are useful, but they are less empirically grounded than the CUMCM set.

## Legacy or Secondary Assets

- `topic_specs.json` is a legacy asset and is not part of the active `v1.2-alpha` workflow.
- `rubric_overlay.json` may be used as optional judging context, not as an active control file.

## Important Boundary

These assets do not define startup flow, topic-letter selection, or scoring-state progression in the active workflow.
