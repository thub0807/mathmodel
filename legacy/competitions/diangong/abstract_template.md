# 电工杯摘要模板 (SEED v0.1)

电工杯摘要无官方分段要求, 但工程类一等奖普遍采用 **4 段式**: 背景 / 方法 / 结果 / 工程建议。
字数 600-1000 (估算), 关键词 3-5 个。

---

## 4 段式骨架

```
【背景段】
{1-2 句: 为什么这个问题重要 + 题目背景 ("随着新能源/储能/电网... 的发展, ...")}
{1 句: 题目核心子问题描述}

【方法段】
{1 句: 整体方法名称 — "本文建立了基于 {模型类} 的 {命名变体} 模型"}
{2-3 句: 各子问题方法概要, 每子问 1 句, 突出建模类型}
{1 句: 求解算法 + 验证方式}

【结果段】(必须 ≥3 个工程量化数值)
{Q1 主要结果: "在场景 X 下, 优化方案使 {指标 1} 降低 {%}, 从 {基线} 降至 {新值} (kW/kWh/...)"}
{Q2 主要结果: "{指标 2} 提升 {%}, 满足 {工程约束}"}
{Q3 主要结果: "灵敏度分析表明, 在工程参数 ±{%} 扰动下, 系统 {鲁棒性陈述}"}

【工程建议段】
{1 句: 主要工程建议}
{1 句: 实施代价 / 时序 / 风险简述}

关键词: {keyword1}; {keyword2}; {keyword3}; {keyword4}
```

---

## 填充示例 (A 题, 电网调度类)

```
针对新能源高渗透率下电网调度面临的弃风弃光与备用容量不足问题, 本文研究了
某省级电网在风光储一体化运行模式下的优化调度策略 (题目 2024 年 A 题)。

本文建立了考虑源荷不确定性的多时间尺度协调调度模型, 综合采用基于场景树的
随机规划处理日前调度 (Q1)、模型预测控制处理日内滚动 (Q2) 与改进 NSGA-II 处理多目标
协同优化 (Q3, Q4)。模型采用 Gurobi 与自实现 NSGA-II 求解, 通过 96 点实测负荷与光伏
出力数据回测验证。

回测结果表明, 优化调度方案使弃风率从基线的 12.4% 降至 5.7% (降低 54%), 备用容量
利用率从 68.3% 提升至 84.1%; 灵敏度分析在光伏渗透率 ±30%、储能配比 ±50%、
负荷增速 ±20% 联合扰动下 (拉丁超立方 n=500), 系统总成本波动控制在 ±8% 以内,
表明方案具备工程鲁棒性。

建议电网公司在试点期 (1-2 年) 部署日前调度模块, 中期 (3-5 年) 引入模型预测控制
模块, 长期配套储能扩容至当前 1.4 倍, 预计可降低年弃风率 5 个百分点以上,
对应直接经济效益约 ¥2.3 亿/年。

关键词: 优化调度; 风光储一体化; 多时间尺度; 灵敏度分析
```

(约 530 字, 可加 1-2 句细节到 700-800 字。)

## 必备 anchor (4 个)
1. ✅ 背景段 + 题目情境
2. ✅ 方法段 + 命名变体 + 求解算法
3. ✅ 结果段 ≥3 个工程量化数值 (含单位)
4. ✅ 工程建议段 含可操作 + 量化收益 + 时序

L1 critic 检查这 4 个 anchor 命中数; ≥ 3 才 pass。
