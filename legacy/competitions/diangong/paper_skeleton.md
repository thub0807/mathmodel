# 电工杯论文骨架 (SEED v0.1)

> 电工杯论文 25-30 页常见, 子问 6-8 个, 工程导向。

```
[封面]                                                   (0.5 页, 自定)
  - 队伍编号 / 题号 / 标题

[摘要 + 关键词]                                          (1 页, 4 段式, 600-1000 字)
  - 背景 / 方法 / 结果 / 工程建议
  - 关键词 3-5

[目录]                                                   (~0.5 页)

§1 问题背景与重述                                        (1.5-2 页)
  1.1 工程背景 (为什么重要)
  1.2 题目重述 (用自己的话)
  1.3 数据概览 (题目附件)

§2 问题分析                                              (1-1.5 页)
  2.1 子问题分解
  2.2 各子问关系 (依赖图 / 复用链)
  2.3 整体技术路线

§3 模型假设与符号说明                                    (1 页)
  3.1 假设 (5-8 条, 每条带工程依据)
  3.2 符号表 (全部带物理含义 + 单位)

§4 数据预处理                                            (1.5-2 页, 电工杯重要章节)
  4.1 数据描述 (字段 / 时间跨度 / 采样)
  4.2 缺失处理 (方法 + 缺失率)
  4.3 异常剔除 (3σ / IQR / 工程上限)
  4.4 预处理后统计描述

§5 子问题求解                                            (12-18 页, 主体)
  5.1 子问题 1 (主线, 4-5 页)
       5.1.1 问题描述 + 工程语境
       5.1.2 模型建立 + 命名变体
       5.1.3 求解算法 + 复杂度
       5.1.4 结果 + 物理意义 + 工程图
  5.2 子问题 2 (主线, 4-5 页)
  5.3 子问题 3 (主线, 3-4 页)
  5.4 子问题 4 (加分, 1-2 页 + 附录代码)
  5.5 子问题 5 (加分, 1-2 页)
  ...

§6 灵敏度与鲁棒性分析                                    (1.5-2 页)
  6.1 工程参数扰动设计 (光伏渗透率 / 储能配比 / 负荷增速)
  6.2 联合扰动结果 (LHS / Sobol)
  6.3 鲁棒区间 + 工程含义

§7 工程实施建议                                          (1-1.5 页, 电工杯加分项)
  7.1 试点 / 中期 / 长期分阶段方案
  7.2 实施代价估算
  7.3 落地风险与应对

§8 模型评价                                              (1 页)
  8.1 优点 (≥3 条具体)
  8.2 缺点 (≥3 条真实, 含改进路线)
  8.3 推广性

§9 参考文献                                              (1-2 页, GB/T 7714)
  - 10-22 条, 电力 / 能源类期刊为主

[附录 A: 主要算法伪代码]
[附录 B: 完整代码 (或 GitHub 链接)]
[附录 C: 补充图表]
```

## 页数预算

| Section | 页数 | 备注 |
|---------|------|------|
| 封面 + 摘要 + 目录 | 2 | |
| 问题背景 + 分析 | 2.5-3.5 | |
| 假设 + 符号 + 数据预处理 | 2.5-3 | 数据预处理是亮点 |
| 子问求解 | 12-18 | 主体, 主线详写 + 加分简写 |
| 灵敏度 | 1.5-2 | 工程参数 + 联合扰动 |
| 工程实施 | 1-1.5 | **电工杯特色, 不可省** |
| 评价 + 参考文献 | 2-3 | |
| 附录 | 2-4 | 代码 + 算法 + 补充图 |
| **Total** | **25-37** | 推荐 27-30 |

## 必备 anchor (stage 9 检查)

- [ ] 摘要 4 段, ≥3 个工程量化数值
- [ ] 数据预处理章节存在 (§4)
- [ ] 题目附件数据全部使用 / 列出
- [ ] 子问主线 ≥3 个, 加分 ≥1 个 (体现分级)
- [ ] 灵敏度参数含工程量纲
- [ ] 工程实施建议章节 (§7)
- [ ] 结论 / 建议含 ≥3 条 numbered actionable
- [ ] 工程惯用图 (单线图 / 时序 / 雷达) 至少 3 类
- [ ] 引用 ≥10 条
- [ ] 附录代码完整或 GitHub 链接
