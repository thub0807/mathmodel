# 电工杯 Phrase Bank (SEED v0.1)

> 工程语境中文学术句式。L1 critic stage 8 anchor 命中数 ≥ N 触发 pass。

---

## 1. 摘要段

### 背景段
- "随着 {新能源 / 电网 / 储能} 的快速发展, {核心问题} 已成为 {电力系统 / 能源行业} 的关键挑战。"
- "针对 {题目情境}, 本文研究了 {核心子问题}, 并提出了 {方法名称}。"

### 方法段
- "本文建立了基于 {模型类} 的 {命名变体} 模型, 综合考虑了 {因素 1}、{因素 2} 与 {因素 3}。"
- "采用 {算法} 求解, 通过 {验证方法} 验证了模型的有效性。"

### 结果段 (含工程数值)
- "在 {场景 1} 下, 优化方案使 {指标} 降低 {百分比}%, 从 {基线值} 降至 {新值} (工程单位)。"
- "灵敏度分析表明, {关键参数} 在 ±{percent}% 扰动下, 系统稳定性保持在 {阈值} 之上。"

### 工程建议段
- "建议电网公司 / 能源企业 {具体建议 1}; 该建议预计可带来 {量化收益}。"

---

## 2. 引言 / 问题重述

- "近年来, {领域} 受到广泛关注, 其核心挑战在于 {挑战}。"
- "现有研究主要采用 {传统方法}, 但当 {条件} 时存在 {局限}。"
- "本文的主要贡献包括: (1) {贡献 1}; (2) {贡献 2}; (3) {贡献 3}。"

## 3. 假设与符号

- "为简化分析, 本文做如下假设, 其工程合理性见 §X。"
- "假设 1: {内容}; 依据 {物理 / 工程 / 历史数据} 支撑。"

## 4. 数据预处理

- "原始数据包含 {N} 条记录, 时间跨度 {时间段}, 采样频率 {频率}。"
- "采用 {方法} 对缺失值进行填补 (缺失率 {ratio}%); 用 3σ 准则识别并剔除异常值 ({count} 条)。"
- "预处理后数据时序曲线见图 X, 统计描述见表 Y。"

## 5. 模型建立

- "设 {变量} ∈ {域} 表示 {物理含义, 单位 kW/kWh/...}。"
- "目标函数为 {经济 / 物理 / 安全} 指标 {表达式}。"
- "约束条件来自 {电网 / 设备 / 调度规则}: 等式约束 (1)-(3); 不等式约束 (4)-(7)。"

## 6. 求解与算法

- "采用 {求解器 / 启发式 / 强化学习} 求解, 算法步骤见算法 1。"
- "在 {硬件配置} 上, 单次求解耗时 {秒}, 满足在线调度需求。"
- "收敛性由 {收敛指标} 在 {代数} 内降至 {阈值} 验证。"

## 7. 灵敏度 / 鲁棒性

- "对 {p1, p2, p3} 联合扰动 (拉丁超立方采样, n=500), 输出指标 {metric} 的 95% 置信区间为 [{lower}, {upper}]。"
- "对工程关键参数 (光伏渗透率 / 储能配比 / 负荷增速) 各扰动 ±20%, 系统稳定性保持。"

## 8. 工程实施 / 落地

- "该方案在 {省/市} 电网试点部署需 {资源}, 预计 {年限} 年回本。"
- "工程上可分阶段实施: 第一阶段 {内容}; 第二阶段 {内容}。"
- "落地风险主要来自 {风险点}, 可通过 {应对} 缓解。"

## 9. 结论与建议

- "本文针对 {核心问题} 建立了 {方法名}, 主要结论如下:"
- "建议电网公司 / 能源企业:"
  1. "{建议 1, 含可操作动作 + 预期效益}"
  2. "{建议 2}"
  3. "{建议 3}"
- "本研究的局限性在于 {局限}; 未来工作可在 {方向} 进一步深入。"

## 10. 工程术语标准化

| 推荐 | 避免 |
|------|------|
| 渗透率 / 出力 / 弃风率 | 占比 / 输出 / 浪费率 |
| 调峰 / 调频 / 备用 | 控制 / 平衡 |
| 负荷曲线 / 时序 | 数据 / 序列 |
| 节点电压 / 潮流 | 电压 / 流量 |
| 储能配比 | 储能比例 |

| Stage | 类别 | 命中数 ≥ |
|-------|------|---------|
| 8 (摘要) | 摘要 4 段 anchor | 3 (背景 + 方法 + 结果 + 建议) |

| 8 (数据预处理) | 数据章节 anchor | 2 |
| 8 (模型) | 模型 setup anchor | 3 |
| 8 (灵敏度) | 灵敏度 anchor | 1 (含工程参数) |
| 8 (工程实施) | 工程落地段 | 1 |
| 8 (结论) | 建议 anchor | ≥3 numbered |
