# Legacy Reference: Final Review and Anti-Pattern Audit

This file now supports `final_review_protocol.md`.

## Knowledge To Keep

- Run an explicit anti-pattern pass before calling the paper complete.
- Review the paper from multiple perspectives when the environment supports it.
- Fix the weakest section, not only the most visible section.
- Confirm figure, table, formula, notation, and unit consistency.
- Check anonymity and PDF status as part of final review.
