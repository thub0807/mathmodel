# 论文语料说明

`references/papers/` 用于存放可选的论文语料参考，主要服务后续知识层升级与语料维护。

## 当前定位

- 它不是 `v1.2-alpha` active workflow 的组成部分。
- 它不是启动流程。
- 正常建模过程中不应要求用户先做年份、题号或选题选择。

## 适用场景

- 后续补充竞赛论文语料
- 更新经验分布、写作模式或反模式知识
- 验证知识层升级是否有足够样本支持

## 使用边界

- 即使这个目录为空，当前 Skill 仍可正常运行。
- 正常 agent 建模应优先使用 `workspace/problem/problem.md`、附件、图片和 active references。
