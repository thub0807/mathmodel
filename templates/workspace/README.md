# Workspace Templates

`templates/workspace/` is organized by artifact scope:

- `project/`: shared project-level files created before or across question work
- `per_question/`: files created inside each question workspace such as `workspace/output/q1/`
- `final/`: final integration, paper, and review files created inside `workspace/output/final/`

Generated-only artifacts such as `paper.pdf`, `results/run.log`, and code directories do not require static content templates.
