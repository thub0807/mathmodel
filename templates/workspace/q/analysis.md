# Analysis 模板

## 文件用途

描述单个 `q*` 的目标、输入、输出、依赖和评价指标。

## 对应 stage

Stage 2 Per-Question Plan

## 必填字段

| 字段 | 填写规则 |
|---|---|
| 子问题目标 | 说明要解决什么 |
| 输入 | 列出题面、附件或前序结果 |
| 输出 | 列出本问应产生的结果 |
| 依赖 | 说明是否依赖其他 `q*` |
| 对应附件 | 列出材料路径 |
| 评价指标 | 说明如何判断结果好坏 |

## 来源字段

`question_index.md`、`problem.md`、`material_index.md`

## 可追溯要求

所有输入和附件必须指向来源文件。

## 禁止空泛表达

不要写“分析题意并建立模型”。必须拆成输入、输出和指标。
