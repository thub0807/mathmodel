# Table 模板

## 文件用途

记录论文候选表格的 Markdown 版本。

## 对应 stage

Stage 5 Figures and Tables

## 必填字段

表名、数据来源、列说明、表格内容、结论。

## 来源字段

结果文件、验证文件、灵敏度文件或附件数据。

## 可追溯要求

每列数值必须能追溯。

## 禁止空泛表达

不要写无来源数字。
